# 📊 Обработка данных CSV файла

Веб-приложение для анализа выручки интернет-магазина по штатам за указанный год.

## 🚀 Основные функции

- 📁 **Загрузка данных** из CSV файла `orders_customers.csv`
- 📅 **Выбор года** для анализа через интуитивный интерфейс
- 🏆 **Топ-10 штатов** по выручке с визуализацией
- 📊 **Детальная таблица** с показателями:
  - № п/п
  - 🏛️ Штат
  - 📦 Количество заказов
  - 💰 Выручка ($)

## 📈 Аналитика

- **💰 Общая выручка** за выбранный год
- **🏆 Выручка топ-10 штатов** 
- **📊 Доля топ-10** в общей выручке (%)
- **📈 Визуальное представление** данных

## 🛠 Технологии

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Обработка данных**: FileReader API
- **Стилизация**: Modern CSS с анимациями
- **Адаптивность**: Mobile-first дизайн

## 🎯 Особенности интерфейса

- ✨ **Интуитивный дизайн** с плавными анимациями
- 📱 **Адаптивная верстка** для всех устройств
- ⚡ **Быстрая обработка** больших объемов данных
- 🎨 **Визуализация данных** через графики и диаграммы
- 🔍 **Интерактивные элементы** для удобного анализа
